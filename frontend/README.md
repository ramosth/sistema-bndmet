### Configuração do Frontend

#### 1. Navegação para Diretório de Frontend

```bash
cd frontend
```

#### 2. Inicialização do Servidor de Desenvolvimento

```bash
npm run dev
```

**Saída de inicialização bem-sucedida:**
```
> frontend@0.1.0 dev
> next dev

  ▲ Next.js 14.2.4
  - Local:        http://localhost:3000
  - Environments: .env.local
```
